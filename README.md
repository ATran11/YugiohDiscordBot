# YugiohDiscordBot
## Description
A Discord Bot I made for my friends for fun to help with our Yu-Gi-Oh discord channel:
* Tells Jokes
* List current pack set and prizing for tournaments
* Assigns roles
* Suggests rock, paper, scissors choices
## Technology used 
* Git Bash
* VS Code
* Discord
